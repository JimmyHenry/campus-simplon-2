JSON
JavaScript Object Notation.
Inventé par Douglas Corckford (le bon Doug').
Format d'échange de plus en plus populaire, JSON c'est bon, mangez-en.
Il permet d'échanger des données structurées entre des applications, quel que soit le langage utilisé pour les coder.
Exemple: en JS, je converti une valeur en JSON et l'envoie vers PHP. PHP pourra décoder cette valeur puis l'interpréter.
